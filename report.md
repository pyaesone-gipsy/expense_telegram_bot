# Chapter 3 Project Report

## Project
Telegram Expense Receipt Bot

## Description
A Telegram bot that accepts receipt screenshots, extracts text using OCR, categorizes expenses automatically, and exports transaction history as CSV.

## Features
- Receive receipt images via Telegram
- OCR extraction with Tesseract
- Automatic category selection for expenses
- Export saved expenses to CSV
