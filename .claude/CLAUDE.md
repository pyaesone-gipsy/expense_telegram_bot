# Expense Receipt Bot — Claude Code Project

This repository contains a Claude Code-enabled Telegram expense receipt bot.

## Project scope
- Receipt image upload via Telegram
- OCR text extraction using Tesseract
- Expense categorization and storage
- CSV export of expense history

## Structure
- `.mcp.json` — project metadata for Claude Code
- `.claude/agents` — agent definitions
- `.claude/skills` — skill definitions
- `.claude/plans` — optional planning workflows
- `bot.py` — Telegram bot implementation
- `requirements.txt` — Python dependencies
- `slides/pitch.md` — project pitch slides
- `report.md` — chapter report
- `README.md` — project overview and setup instructions
