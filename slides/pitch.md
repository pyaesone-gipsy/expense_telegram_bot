# Slide 1: Problem
People lose track of spending when receipts are stored as photos. Manual entry is slow and error-prone.

---

# Slide 2: Solution
A Telegram bot that accepts receipt screenshots, reads the text, categorizes expenses, and saves records automatically.

---

# Slide 3: Key Features
- Photo receipt upload
- OCR text extraction
- Automatic expense categorization
- CSV export for budgeting

---

# Slide 4: Workflow
1. Send receipt image to bot
2. Bot reads the slip
3. Bot identifies amounts and categories
4. Use `/export` to download expense data

---

# Slide 5: Benefits
- Saves time
- Improves expense visibility
- Makes export easy for accounting
- Works on Telegram without installing extra apps

---

# Slide 6: Next Step
Run the bot, test with a real receipt photo, and extend categories for local shops and utilities.
